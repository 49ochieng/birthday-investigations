# Martin's Photo Evidence

Place Martin's birthday photos in this folder using these exact filenames so the site picks them up automatically:

- `martin-1.jpg`
- `martin-2.jpg`

If a file is missing, the site will gracefully display:

> "Photo evidence pending upload. The investigation continues."

You can add more images and reference them from `src/data/martin.ts` (the `images` array) if you want additional photos beyond the two defaults.
